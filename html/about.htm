About this tool


This tool has been developed by dodo182 and marcosox.
The goal of this tool is to show some informations about the road accidents analyzed during the BigData course final assignment.
This tool has been realized as an assignment for the InfoVis course held by Maurizio Patrignani at Roma Tre University (http://www.dia.uniroma3.it/~infovis) and uses the following technologies:

  • D3.js
  • Bootstrap
  • Jquery
  • Google Maps
